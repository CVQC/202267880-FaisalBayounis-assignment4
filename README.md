## 📌 Project Description

A fully polished, production-ready personal portfolio website built with HTML5, CSS3, and vanilla JavaScript. This final version builds on Assignment 3 by adding UI/UX enhancements, performance optimizations, professional code quality, and a live deployment — resulting in a complete, real-world-ready web application.

---

## ✨ What's New in Assignment 4

- 🎨 **Scroll Progress Bar** — visual indicator showing how far down the page the user has scrolled
- 🔢 **Animated Number Counters** — skills and stats animate into view on scroll using IntersectionObserver
- 🌈 **Gradient Hero Text** — polished name styling with smooth CSS gradient effect
- 🌙 **Full Dark Mode Coverage** — all new components fully support dark/light theme toggle
- ♿ **Accessibility Improvements** — added `aria-label` attributes, semantic HTML, and keyboard navigation support
- ⚡ **Performance Optimizations** — `loading="lazy"`, `preconnect`, `dns-prefetch`, `defer`, and debounced scroll events
- 🚀 **Live Deployment** — deployed to Vercel with automatic redeployment on every GitHub push
- 🧹 **Code Quality** — consistent formatting, indentation, naming conventions, and removed all debug logs

## 🚀 Features

- 🌙 Dark / Light mode toggle with localStorage persistence
- 👋 Visitor name input — saved and restored on every visit
- 🗂️ Project filter (All / COE / CS) + Sort (A→Z / Z→A)
- 📂 Expand / Collapse skill cards
- 🐙 Live GitHub repositories fetched from GitHub API
- 💬 Random inspirational quotes from DummyJSON API
- 🎵 iTunes Music Search with 30-second audio preview player
- 📬 Contact form with full validation and confirmation message
- ✨ Scroll-reveal animations, hover effects, and fade-in transitions
- 📊 Scroll progress bar at the top of the page
- 🔢 Animated number counters on scroll
- ⏱️ Live visit duration timer in the footer

## 🧭 How to Use the Website

| Feature | How to Use |
|---------|-----------|
